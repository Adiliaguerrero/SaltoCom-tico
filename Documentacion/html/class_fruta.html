Detailed Description

Representa una fruta recolectable que restaura vida al jugador al entrar en contacto.

Este script debe estar adjunto a un objeto con un Collider2D marcado como Trigger. Se espera que el jugador tenga un componente PlayerController con el método RestaurarVidaParcial.

Asigna este script a una fruta y asegúrate de configurar cantidadVida desde el Inspector.
