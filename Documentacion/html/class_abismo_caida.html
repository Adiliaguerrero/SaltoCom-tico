Detailed Description

Controla el comportamiento del abismo de caída. Si el jugador entra en contacto con esta zona, el personaje muere y se muestra el panel de Game Over.

Esta clase debe ser añadida a un GameObject con un Collider2D marcado como "Trigger". Se utiliza OnTriggerEnter2D(Collider2D) para detectar la colisión del jugador.
