Detailed Description

Gestiona el comportamiento de un pájaro enemigo que se mueve horizontalmente y daña al jugador al contacto.

El pájaro vuela de un lado a otro en un rango definido y cambia de dirección tras un tiempo de espera. Si colisiona con el jugador, le inflige daño usando el método RecibeDanio.
